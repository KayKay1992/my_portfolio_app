SETTING UP OUR LIVE PREVIEW IN PRISMIC,

please note to make sure your slice machine is running if not running start it with this command npm run slicemachine.
To set up live preview in prismic go to homepage page, click on live preview, put your project url e.g http://localhost:3000/slice-simulator always make sure you include the lice-simulator extension to your project url, then click save.
